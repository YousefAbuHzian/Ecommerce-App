# Ecommerce-App
A full-featured cross-platform e-commerce application built end-to-end, with an ASP.NET Core REST API backend and a Flutter mobile frontend.

✨ Features
🛒 Product browsing & search
🧺 Cart management
📦 Order placement & tracking
🔔 Push notifications (Firebase FCM)
🎯 Promotional banners
🌐 Bilingual support — English & Arabic
